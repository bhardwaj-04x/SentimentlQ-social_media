01 Abstract

This project presents SentimentIQ, a web-based AI system that performs sentiment analysis on social media text using an ensemble of three models: VADER, TextBlob, and a trained Machine Learning classifier (TF-IDF + Logistic Regression).

The final sentiment label is determined by majority vote across all three engines, making the system robust against individual model errors. The system supports both single-text and bulk analysis (up to 50 texts), with interactive visualizations.

02 Objectives
  • Build a real-time sentiment analysis system for social media text
  • Combine rule-based (VADER, TextBlob) and ML-based approaches into an ensemble
  • Develop a full-stack web application using Flask and Bootstrap 5
  • Provide bulk analysis capabilities with visual summary charts
  • Expose a REST API for programmatic access to the sentiment engine

  • Linguistic patterns
  • Polarity: −1 to +1

  • Confidence probability
